# OpenJDK Valhalla Demo
Demo repository of [OpenJDK Valhalla](https://openjdk.java.net/projects/valhalla/) features

To run the demo:
```bash
export JAVA_HOME=/path/to/openjdk-valhalla
$JAVA_HOME/bin/java -jar jvisualbook.jar
```

Then visit http://localhost:8080/ and select the slide mode to view the slides.
